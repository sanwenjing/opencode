此技能指导创建独特、生产级前端界面，避免通用"AI 泥浆"美学。实现真正可工作的代码，对美学细节和创意选择给予卓越关注。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能包括关于目的、受众或技术约束的上下文。

 设计思维

在编码之前，了解上下文并承诺大胆美学方向：
- **目的**：此界面解决什么问题？谁使用它？
- **语调**：选择极端：野蛮极简、极繁混乱、复古未来、有机/自然、奢华/精致、好玩/玩具化、编辑/杂志、野兽派/原始、装饰艺术/几何、柔和/粉彩、工业/实用等。有许多风格可供选择。将这些用作灵感，但设计一个忠于美学方向的。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让它难忘？人们会记住的那一件事是什么？

**关键**：选择清晰的概念方向并精确执行它。大胆极繁主义和精致极简主义都有效 - 关键是意图性，而不是强度。

然后实现可工作的代码（HTML/CSS/JS、React、Vue 等），这些代码是：
- 生产级和功能性的
- 视觉上引人注目和难忘的
- 具有清晰美学观点的连贯性
- 在每个细节上都经过精心完善

 前端美学指南

专注于：
- **字体排印**：选择美观、独特和有趣的字体。避免像 Arial 和 Inter 这样的通用字体；选择提升前端美学的独特选择；意想不到、有性格的字体选择。将独特的显示字体与精致的正文字体配对。
- **颜色与主题**：坚持连贯美学。使用 CSS 变量保持一致性。主导色配锐强调色胜过胆小、均匀分布的调色板。
- **动作**：使用动画制作效果和微交互。优先使用仅 CSS 解决方案进行 HTML。当可用时使用 Motion 库进行 React。专注于高影响时刻：一个精心编排的页面加载与交错显示（animation-delay）比分散的微交互创造更多愉悦。使用滚动触发和悬停状态来惊喜。
- **空间构图**：意外布局。不对称。重叠。对角流动。网格破坏元素。慷慨的负空间或受控的密度。
- **背景与视觉细节**：创造氛围和深度，而不是默认为纯色。添加与整体美学匹配的上下文效果和纹理。应用创意形式如渐变网格、噪声纹理、几何图案、分层透明度、戏剧性阴影、装饰边框、自定义光标和颗粒覆盖。

永不使用通用 AI 生成的美学，如过度使用的字体系列（Inter、Roboto、Arial、系统字体）、陈词滥调的颜色方案（特别是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定特征的千篇一律设计。

创造性解释并做出真正为上下文设计的意外选择。没有设计应该是相同的。在明暗主题、不同字体、不同美学之间变化。世代间永不收敛于常见选择（例如 Space Grotesk）。

**重要**：将实现复杂度与美学愿景匹配。极繁主义设计需要具有广泛动画和效果的精细代码。极简主义或精致设计需要克制、精确和对间距、字体排印和微妙细节的仔细关注。优雅来自良好执行愿景。
