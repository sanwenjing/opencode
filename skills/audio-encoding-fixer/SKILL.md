## Audio Encoding Fixer - 音频文件编码修复工具

### 功能概述

这是一个专业的音频文件编码修复工具套件，专门解决音乐文件的标题和艺术家名称乱码问题。无论你的音乐库是从哪里来的，如果出现了乱码字符（如），这个工具都能帮你修复。

### 支持的音频格式

- MP3
- WMA
- WAV
- FLAC
- M4A
- OGG

### 包含的工具脚本

#### 1. fix_encoding.py - 编码检测脚本
检测音频文件名中的编码问题，自动识别乱码字符，并提供重命名建议。

#### 2. audio_encoding_fix_tool.py - 综合修复工具（主菜单界面）
提供交互式主菜单，集成所有修复功能：
- 检查文件名编码问题
- 修复音频文件ID3标签
- 手动文件名编码修复助手
- 批量重命名工具

#### 3. fix_audio_tags.py - ID3标签修复脚本
使用 mutagen 库修复音频文件的ID3标签（标题、艺术家等元数据）中的乱码问题。

#### 4. simple_audio_tag_fixer.py - 简化标签扫描工具
扫描音频文件的所有ID3标签，显示当前标签内容，检测乱码问题，并提供修复建议。

#### 5. fix_direct.py - 直接文件名修复脚本
直接修复包含乱码字符的音频文件名，支持自动修复和批量处理。

#### 6. windows_encoding_fix.py - Windows编码修复
针对Windows系统的特殊编码问题提供解决方案，使用Windows命令行获取准确的文件名。

#### 7. create_fix_tools.py - 修复工具创建器
自动创建所有修复脚本和批处理文件，一键生成完整的修复工具集。

### 使用方法

#### 快速开始
```bash
python scripts/audio_encoding_fix_tool.py
```

#### 检测文件名编码问题
```bash
python scripts/fix_encoding.py
```

#### 修复音频标签
```bash
python scripts/fix_audio_tags.py
```

#### 直接修复文件名
```bash
python scripts/fix_direct.py
```

#### 扫描所有标签
```bash
python scripts/simple_audio_tag_fixer.py
```

### 技术特性

- **编码自动检测**: 支持 GBK、GB2312、GB18030、UTF-8、Big5、CP936 等多种编码格式
- **智能修复算法**: 自动识别中文字符，确保修复结果准确无误
- **批量处理能力**: 支持批量扫描和修复整个音乐目录
- **安全操作**: 提供重命名预览和确认步骤，避免误操作
- **UTF-8 控制台支持**: 所有脚本都包含 Windows 控制台 UTF-8 编码设置
- **依赖自动安装**: 自动检测并安装必要的依赖库（mutagen、chardet）

### 依赖库

- `mutagen` - 音频文件元数据处理
- `chardet` - 字符编码检测（部分脚本使用）

### 工作原理

1. **编码检测**: 通过检测文件名或标签中的乱码字符（），识别编码问题
2. **编码转换**: 尝试从可能的原始编码（如 GBK）转换为正确的编码（UTF-8）
3. **中文验证**: 验证转换结果是否包含有效的中文字符，确保修复准确
4. **安全重命名**: 在确认目标文件不存在的情况下执行重命名操作

### 注意事项

- 修复前建议备份原始文件
- 对于无法自动修复的文件，可以手动提供正确名称
- Windows 系统下可能需要在命令行中使用 `chcp 65001` 命令设置 UTF-8 编码

### 适用场景

- 从旧系统迁移的音乐文件出现乱码
- 下载的音乐文件名显示为乱码
- 音频播放软件显示的标题/艺术家名称乱码
- 需要将不同编码格式的音乐库统一为 UTF-8
